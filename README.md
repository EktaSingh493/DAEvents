# DAEvents
DA-IICT collage app to share events held by clubs and committees

This is an android application for DA-IICT collage to let it's clubs and committees share their events to the registered users.

## Features:
1. Shows the list of all upcomming and past events and their details
2. Let the user like or set reminder for specific events
3. Committee members can add new events
4. User can hide events he/she is not interested in
5. Beautiful UI
